# Lawnicons

[Lawnicons](https://lawnchair.app/lawnicons) is a free, open source icon pack that provides outlined icons that match Google's themed icons.

For more information, view the repository page linked above.&#x20;

### Prerequesties

* The Lawnicons app installed

### How to use Lawnicons in Lawnchair

Lawnicons can be used in two different ways: as an icon pack, and as a themed icon source.

#### As an icon pack

1. Navigate to **Home settings** > **General** > Icon pack
2. On the list that shows, tap **Lawnicons**.

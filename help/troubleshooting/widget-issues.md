# Widget issues

Widget behavior on Android can sometimes be inconsistent due to variations in how different apps, launchers, and device manufacturers handle them.

This page addresses common problems you might encounter with widgets in Lawnchair, alongside solutions.

<details>

<summary>Widget not appearing or crashing during setup</summary>

This issue can occur when adding widgets that have a setup screen, especially on older Android versions.

To fix this, follow these steps:

1. Temporarily change your default launcher to your system's default launcher (e.g., Pixel Launcher, One UI Home).
2. Open Lawnchair.
3. Attempt to add the widget you want to use.
4. Re-set Lawnchair as the default launcher.

</details>

<details>

<summary>Widget data not updating</summary>

If a widget's information (e.g., weather, calendar events) is not updating, follow these steps:

1. Resize the widget to a different size, then return it to its original size.

</details>

<summary>All widgets disappeared from the home screen</summary>

This can sometimes happen when the home screen is not active for some time. To fix this, follow these steps:

1. Long press on the home screen, then tap **Home settings**.
2. Tap **More** <i class="fa-ellipsis-vertical">:ellipsis-vertical:</i> >  <i class="fa-arrow-rotate-left">:arrow-rotate-left:</i> **Restart Lawnchair.**

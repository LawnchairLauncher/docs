# Choosing a version

Lawnchair is available through different release channels, each with different levels of stability and feature availability.

Use the table below to determine which version is appropriate for you.

<table><thead><tr><th width="154">Release channel</th><th width="193">Intended users</th><th>Stability</th><th>Characteristics</th></tr></thead><tbody><tr><td><a href="https://lawnchair.app/downloads">GitHub</a></td><td>Most users</td><td>Generally stable, suitable for daily use</td><td>A balance of new features and reliability. Released periodically after testing.</td></tr><tr><td>Nightly</td><td>Testers, enthusiasts, users needing a specific fix</td><td>Potentially unstable. May contain bugs.</td><td>Built automatically with the latest code changes. Has a built-in updater. Installs as a separate app from the Beta version.</td></tr><tr><td><a href="https://play.google.com/store/apps/details?id=app.lawnchair.play">Play Store</a></td><td>Users who prefer automatic updates via Google Play</td><td>Same with Official Beta</td><td>Releases for the Play Store often follow Official Beta releases but can be delayed by 1-2 weeks due to platform review processes. Uses a different package name from GitHub versions.</td></tr></tbody></table>

### Summary

* For a stable experience, begin with the GitHub or Play Store version.
* If you want the newest features and are comfortable with potential bugs, use the **Nightly** builds.

# Install and setup

This guide will walk you through installing Lawnchair and setting it as your default home screen.

{% stepper %}
{% step %}
### Install the app

Download and install the version you chose in [choosing-a-version.md](choosing-a-version.md "mention").
{% endstep %}

{% step %}
### Set Lawnchair as default home app

After installation, you must tell Android to use Lawnchair as your main home screen.

1. Long press on the home screen, then tap **Home settings**
2. In Home settings, tap the card that states:
   * **To access shortcuts and additional features, set Lawnchair as your default launcher**
3. In the screen that appears, select **Lawnchair** from the list.
{% endstep %}
{% endstepper %}
